# Agentic AI Assistant

## Overview
An AI agent capable of performing multi-step tasks using Large Language Models (LLMs) with reasoning and tool usage.

## Features
- Task decomposition using LLMs
- Multi-step reasoning
- Integration with external tools (APIs)
- Autonomous decision making

## Tech Stack
- Python
- LangChain Agents / CrewAI
- OpenAI API

## Example Tasks
- Summarize news articles
- Fetch and process data from APIs
- Automate repetitive workflows

## How it works
1. User provides a task  
2. Agent breaks task into smaller steps  
3. Uses tools/APIs when needed  
4. Generates final output
